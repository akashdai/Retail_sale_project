# Retail Sales SQL Project

## Project Overview

This project analyzes retail sales data using PostgreSQL and SQL queries.

The project includes:
- Database creation
- Data cleaning
- Null value handling
- Data exploration
- Business problem solving using SQL

---

- PostgreSQL
- SQL
- pgAdmin 4
- GitHub
- VS Code

- age
- category
- quantity

## SQL Analysis & Business Problems Solved

### Data Exploration
- Total sales count
- Unique customer count
- Unique product categories

### Business Problems
1. Sales on a specific date
2. Clothing transactions with quantity filters
3. Total sales by category
4. Average customer age by category
5. Transactions with high sales
6. Gender-wise transaction analysis
7. Monthly average sales analysis
8. Top 5 customers by sales
9. Unique customers per category
10. Shift-wise order analysis
